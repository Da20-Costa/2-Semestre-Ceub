Exemplo 8 HTML - Uso das tags <ol>, <ul> e <li>

As tags <ol>, <ul> e <li> representam listas em html, e são utilizadas para as mais diversas finalidades. A <ol> representa uma lista ordenada, já a tag <ul> representa uma não ordenada.

Em ambas as listas, para representar os elementos delas, usamos a tag <li>

Além disso, estas listas podem conter outras listas de forma aninhadas.
