Uso dos métodos (push, pop, sort, lastIndexOf e IndexOf)

  • push(elemento)
    Adiciona um ou mais elementos ao final do array. É um método mutável (altera o array original).
    Retorno: O novo tamanho (comprimento) do array.
    Exemplo: arr.push('novo item');
  • pop()
    Remove o último elemento do array. Também altera o array original.
    Retorno: O elemento que foi removido.
    Exemplo: let removido = arr.pop();
  • sort()
    Ordena os elementos do array localmente e retorna o array ordenado.
    Atenção: O padrão é a ordenação alfabética (strings). Para ordenar números corretamente, é necessário passar uma função de comparação.
    Exemplo: arr.sort((a, b) => a - b); (Ordenação numérica crescente)
  • indexOf(elemento)
    Retorna o primeiro índice em que o elemento pode ser encontrado no array.
    Retorno: O número do índice (ex: 0, 1, 2...) ou -1 se o elemento não existir.
    Exemplo: let posicao = arr.indexOf('item');
  • lastIndexOf(elemento)
    Retorna o último índice em que um elemento pode ser encontrado no array (faz a busca de trás para frente).
    Retorno: O número do índice ou -1 se não encontrado. Útil quando há valores duplicados no array.
    Exemplo: let ultimaPosicao = arr.lastIndexOf('item');
