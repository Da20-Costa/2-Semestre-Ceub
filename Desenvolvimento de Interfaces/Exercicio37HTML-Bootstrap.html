Bootstrap

O bootstrap é um framework front-end e fornece componentes e ferramentas para facilitar a criação de sites responsivos. Ele foi desenvolvido pelo Twitter e é muito útil para aqueles que precisam de aplicar CSS a documentos html.

Container

Container é um classe fundamental do bootstrap, usada para delimitar e alinhar o conteúdo na página, definindo limites para o layout e garantidno o espaçamento adequado entre os elementos.

O seletor de classe container pode ser aplicado de duas formas: container ou container-fluid. Sendo assim, podemos dizer que o container, quando aplicado delimita larguras fixas nas laterais do documento html. Já o container-fluid, quando aplicado, ocupa 100% do layout da tela disponível.
